## Como usar?

### Dolar

Você pode chamar as cotações via linha de comando. Por exemplo:


```bash
{{ commands.run }} dolar-comercial
```

O formato retornado será bem simples. 

```
┏━━━━━━━━━━━━━━━┓
┃ USD Comercial ┃
┡━━━━━━━━━━━━━━━┩
│ R$ 5.1125     │
└───────────────┘
```
